【DP专项】005.Longest Palindromic Substring

一、题意

题目链接: 005.Longest Palindromic Substring

1.Description

【输入】
给定一个长度为n的字符串s；
【输出】
求出s的某个最长回文子串。可能存在多个，任意返回一个即可。

2.Example

Input: “babad”
Output: “bab”
Note: “aba” is also a valid answer.

3.Corner Case

  1. n <= 1000
  2. 若n为0，返回空串；
  3. 若n为1，返回s；

二、题解

思路

  1. 同样，可以用brute force，找出判断所有的子串，看其是否为回文串。
    找出最大的长度，以及开始的索引startIndex。

Solution-1：Dynamic Programming

  1. 状态定义：dp[i][j]表示：子字符串s[i]…s[j]是否为回文串，不是的话值为0，是的话值为其长度。

  2. Code(https://leetcode.com/submissions/detail/292341253/)
    时间复杂度：O(n * n)
    空间复杂度：O(n * n)

  3. Code(https://leetcode.com/submissions/detail/292065399/)
    时间复杂度：O(n * n)
    空间复杂度：O(n)

Solution-2：中心扩展法

  1. 记录最长回文子串的长度；记录最长回文子串的起始索引；

  2. 有以下两种情况：
    当len1 > len2时，中心点为(i)，长度为len1，所以起始索引为：i - len1 / 2
    当len2 >= len1时，中心点为(i, i + 1)，长度为len2，所以起始索引为：i + 1 - len2 / 2

  3. Code(https://leetcode.com/submissions/detail/292349317/)
    时间复杂度：O(n * n)
    空间复杂度：O(1)
