【DP专项】063.Unique Paths II

一、题意

题目链接: 063.Unique Paths II

1.Description

【输入】
给定一个m * n的矩阵。元素都是0或者1。1代表着该位置有障碍物，0代表没有障碍物。
【输出】
求出，从left-top到right-bottom，一共有多少种不同的路径。
在任意位置上，只能往下或者往右走一步。

2.Example

Input:
[
[0,0,0],
[0,1,0],
[0,0,0]
]
Output: 2
Explanation:
There is one obstacle in the middle of the 3x3 grid above.
There are two ways to reach the bottom-right corner:

  1. Right -> Right -> Down -> Down
  2. Down -> Down -> Right -> Right

3.Corner Case

  1. m或者n为0时，答案为0。

二、题解

Solution-1：Dynamic Programming

  1. 状态定义：dp[i][j]：到达位置(i,j)上，有多少种不同的方法。

  2. 转移方程：dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
    若grid[i][j] = 1，则dp[i][j] = 0

  3. 边界条件：
    a.若grid[0][0] = 1，则答案为0。否则的话，dp[0][0] = 1。
    b.若grid[m - 1][n - 1] = 1，则答案为0。
    c.当i为0时，dp[0][j] = 0/1。当且仅当dp[0][j - 1] == 1 && grid[0][j] == 0时，dp[0][j] = 1。
    d.当j为0时，dp[i][0] = 0/1。当且仅当dp[i - 1][0] == 1 && grid[i][0] == 0时，dp[i][0] = 1。

  4. 求解顺序：从(0,0)开始计算，到(m - 1, n - 1)停止。

5.Code(https://leetcode.com/submissions/detail/261862495/)
时间复杂度：O(m * n)
空间复杂度：O(m * n)

6.Code(https://leetcode.com/submissions/detail/290428618/)
时间复杂度：O(m * n)
空间复杂度：O(n)
