《程序员代码面试指南》-最长公共子串

发表于 2020-06-16 分类于 数据结构与算法 ， 《程序员代码面试指南》

一、题意

0、题目链接

《程序员代码面试指南》-最长公共子串

1、Description

【输入】
给定两个字符串text1，text2，长度分别为n1，n2；
【输出】
找出text1和text2的LCs(Longest Common Substring)；
LCs：最长公共子串
如果没有LCs，则输出-1；
【要求】
无；

2、Example

输入
1AB2345CD
12345EF
输出
2345

3、Corner Case

  1. 1 <= n1、n2 <= 5000

二、题解

0、思考

  1. 类似于1143.Longest Common Subsequence；

  2. LCs，本题中的‘s’指的是子串，而非子序列；子串要求连续；

1、Solution-1：Dynamic Programming

  1. 【化零为整-状态表示】
    dp(i, j)：基于经验的表示；
    代表的集合：text1[0…i]与text2[0…j]的所有公共子串的集合；且这些子串的最后一个字符c == text1[i] == text2[j]；
    集合的属性：求Max值；

  2. 【化整为零-状态计算】
    考虑dp(i, j)可以划分为几部分集合；
    划分原则：不遗漏；不重复（Min/Max允许重复）；
    text1[i] == text2[j]：集合不为空，dp[i][j] = 1 + dp[i - 1][j - 1]；
    text1[i] != text2[j]：集合为空，dp[i][j] = 0；

  3. 计算顺序：
    计算dp[i][j]需要考虑左上角元素；所以：
    i从0开始，到n1 - 1结束；
    j从0开始，到n2 - 1结束；

  4. 边界值：
    若i < 0 || j < 0的时候，dp[i][j] = 0；

  5. 最终答案：根据二维数组dp中最大的元素maxLen，以及endPos，输出LCs；

  6. Code(https://www.nowcoder.com/profile/876389328/codeBookDetail?submissionId=79324000)
    AC
    时间复杂度：O(n1 * n2)
    空间复杂度：O(n1 * n2)

  7. 空间优化-1：由于计算第i行的时候，只需要第i - 1行（左上角元素）即可；
    可以将O(n1 * n2)的空间复杂度降低到O(n)，这里n = min(n1, n2)；

  8. 空间优化-2：由于计算第i行的时候，只需要其第i - 1行（左上角元素）即可；
    可以使用斜线遍历的方法，将O(n1 * n2)的空间复杂度降低到O(1)；
    起点：右上角，终点：左下角；
