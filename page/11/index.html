【DP专项】509.Fibonacci Number

一、题意

题目链接: 509.Fibonacci Number

1.Description

【输入】
给定一个非负整数n。
【输出】
求第n项的Fibonacci值。
F(0) = 0, F(1) = 1
F(N) = F(N - 1) + F(N - 2), for N > 1.

2.Example

Input: 4
Output: 3
Explanation: F(4) = F(3) + F(2) = 2 + 1 = 3.

3.Corner Case

  1. n为0时，答案为0
  2. n为1时，答案为1
  3. 0 ≤ n ≤ 30

二、题解

Solution-1：Dynamic Programming(Iterative)

1.回忆【070.Climbing Stairs】，本质上就是求Fibonacci数列的第n项。
只不过base case需要特定考虑。

2.Code(https://leetcode.com/submissions/detail/282415011/)
时间复杂度：O(n)
空间复杂度：O(1)
