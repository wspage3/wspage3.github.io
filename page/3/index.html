【链表专项】082.Remove Duplicates from Sorted List II

一、题意

题目链接: 082.Remove Duplicates from Sorted List II

1.Description

【输入】
给定一个单链表，头为head。该单链表为有序的（升序）。
【输出】
将上述单链表中value出现多次的元素都去除（一个也不留），仅出现一次的元素保留。

2.Example

Input: 1->2->3->3->4->4->5
Output: 1->2->5

3.Corner Case

  1. 链表为NULL时，无需处理。
  2. 链表为单节点时，无需处理。

二、题解

Solution-1：Iterative Solution

1.回忆【203.Remove Linked List Elements】中，我们遍历链表来寻找所有value为target的节点，并将其删除。
使用一个dummyHead接在链表的前面，初始化cur指向dummyHead。依次检查cur->next是否满足题意……直到cur为空。

2.回忆【083.Remove Duplicates from Sorted List】中，我们遍历链表来寻找所有value重复的节点，保留第一个，将后面的删除。

3.本题中，也使用一个dummyHead接在链表前面。因为链表中第一个元素同样可能是满足题意的，即需要删除的。
pre指针初始化指向dummy。
cur指针初始化执行head。

4.外层循环：当cur非空的时候，考虑cur。
内层循环：考虑cur紧接着的后面若干个元素，其value是否与cur->value相等。相等的话，删除cur后继的并且值相等的元素。
同时置flag为true，表示当前cur也需要删除。

时间复杂度：O(n)
空间复杂度：O(1)
