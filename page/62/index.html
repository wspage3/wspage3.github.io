【二叉树专项】104.Maximum Depth of Binary Tree

一、题意

题目链接: 104.Maximum Depth of Binary Tree

1.Description

【输入】
给定一棵二叉树。
【输出】
找出其最大的深度。
最大深度也就是求：从root到所有的leaf中，路径上最多包含几个节点。
也就是求：这棵树有几层？

2.Example

Input: [3,9,20,null,null,15,7]
Output:
3

3.Corner Case

  1. 树为空时，0。
  2. 树仅包含一个node时，1。

二、题解

Solution-1：Recursive solution(DFS)

1.遍历顺序为dfs的顺序

2.对于任一节点来说：
询问两儿子（递归调用）
计算出来（计算两个答案的最大值，再加一）
上报给父亲（返回答案）

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276168238/)

Solution-2：Iterative solution using Queue(BFS)

1.遍历顺序为bfs的顺序

2.在bfs中，我们每次得到树的某一层的所有几点。
所以只要统计出现过几层，那么答案就有了。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276171830/)
