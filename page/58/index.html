59.Spiral Matrix II

一、题意

题目链接: 59.Spiral Matrix II

1.Description

【输入】
给定一个正整数n。
【输出】
构造一个1 ~ n ^ 2的方阵，它的螺旋序列为1 ~ n。

2.Example

Input: 3
Output:
[
[ 1, 2, 3 ],
[ 8, 9, 4 ],
[ 7, 6, 5 ]
]

3.Corner Case

1.n为正整数。

二、题解

Solution-1：Layer by Layer, One by One

1.根据54.Spiral Matrix的启示，
用同样的方法，从外层到内层，每一层按照top -> right -> bottom -> left的顺序，将当前数字写入到方阵中。

2.由于此时是方阵，我们先考虑a点和b点不同的情况。
设左上角的点a为(r1, c1)，设右下角的点b为(r2, c2)。
初始值a为(0, 0)，b为(n - 1, n - 1)
显然，当a，b两点不同的时候，最小为2 * 2的正方形;
所以循环结束的条件为r1 < r2, c1 < c2

3.遍历完每一层后，需要考虑下一层，即：
r1++;
c1++;
r2–;
c2–;

4.对于每一层，把四条边（r1 < r2, c1 < c2）上的路径走一次即可：
不考虑r1 == r2，c1 == c2的情况。
top：i = r1; j = [c1, c2]
right：i = [r1 + 1, r2]; j = c2
bottom：i = r2; j = [c2 - 1, c1]
left：i = [r2 - 1, r1 + 1]; j = c1

5.两种情况考虑：
当循环最后一次是2 * 2的正方形时，所有元素都遍历完成。
当循环最后一次是3 * 3的正方形时，还剩一个1 * 1的需要遍历。

6.temp代表当前数字，从0开始，每次写入的时候，先自增，后赋值。

时间复杂度：O(n * n)
空间复杂度：O(1) // 不包含返回的结果二维方阵
