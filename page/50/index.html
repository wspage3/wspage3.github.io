【BST专项】108.Convert Sorted Array to Binary Search Tree

一、题意

题目链接: 108.Convert Sorted Array to Binary Search Tree

1.Description

【输入】
给定一个长度为n的数组，数组是升序排列的。
【输出】
将这个数组转化为一个高度平衡的（height-balanced）二叉搜索树（BST）。

2.Example

Input: [-10,-3,0,5,9]
Output: [0,-3,9,-10,NULL,5,NULL] 答案不唯一。

3.Corner Case

  1. n = 0时，树为NULL。
  2. n = 1时，树为单节点。

二、题解

Solution-1：Recursive Solution

1.给定一个长度为n的数组nums，要将其转化为高度平衡的BST。
a.取数组中间元素作为root；
b.递归将其左边所有元素构成一棵较小的高度平衡的BST。作为root的左子树。
c.递归将其右边所有元素构成一棵较小的高度平衡的BST。作为root的右子树。

2.结束条件：若数组长度为1，则将其作为一个单节点。

3.不管n是奇数还是偶数，对于任一节点，其左右子树的节点数不会大于1。保证了height-balanced。

时间复杂度：O(n)
空间复杂度：O(n)

4.Code(https://leetcode.com/submissions/detail/277835456/)

Solution-2：Recursive Solution with Optimization

1.优化：递归调用的时候，避免：构建子数组，将子数组作为参数传递。
而是直接传递索引即可。

时间复杂度：O(n)
空间复杂度：O(n)
