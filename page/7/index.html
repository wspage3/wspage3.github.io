1009.Complement of Base 10 Integer

1、Description

【输入】
给定一个非负整数num；
【输出】
求出num的“补数”；
“补数”规则：将num的二进制形式，每位都取反后，形成的数；

无；

2、Example

Output: 2

002.Add Two Numbers

1、Description

【输入】
给定两个非空链表l1、l2；分别代表一个非负整数；
链表形式为：每个结点代表一位；头结点代表最低位（个位），尾结点代表最高位；
例如：[2 -> 4 -> 3 -> NULL]，代表整数：342；
【输出】
计算这两个整数的和，并且以上述形式返回；
【要求】
无；

2、Example

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8
Explanation: 342 + 465 = 807.
