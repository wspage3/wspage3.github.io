【DP专项】120.Triangle

一、题意

题目链接: 120.Triangle

1.Description

【输入】
给定一个二维数组。代表着一个三角形。
【输出】
求出一条从顶点到最底层任意一点的路径，使得这条路径上的所有结点的和最小。

2.Example

[
[2],
[3,4],
[6,5,7],
[4,1,8,3]
]
The minimum path sum from top to bottom is 11 (i.e., 2 + 3 + 5 + 1 = 11).

3.Corner Case

  1. m为1，n为1时，答案就是triangle[0][0]
  2. m为2，n为2时，答案就是triangle[0][0] + min(triangle[1][0], triangle[1][1])
  3. m与n相等

二、题解

Solution-1：Dynamic Programming(Iterative)

  1. 状态定义：dp[i][j]：从最低端，到达位置(i, j)的所有路径中，最短的那条路径对应的和。答案就是dp[0][0]。

  2. 转移方程：dp[i][j] = min(dp[i + 1][j], dp[i + 1][j + 1]) + triangle[i][j]

  3. 边界条件：当i == n - 1时，dp[i][j] = triangle[i][j]

  4. 求解方向：从倒数第二层开始计算，即i从n - 2开始，到0结束。
    这样的【Bottom-Up】的求解方向，使得遍历完，结果就是dp[0][0]了。
    如果采用【Top-Down】的求解方向，即i从1开始，到n - 1结束。这样的话，需要考虑i - 1，以及j - 1是否存在的问题，即多了if语句判断。
    而且，最后需要找到第n - 1层的最小值。

  5. 空间优化：不另开dp数组，复用triangle数组即可。

  6. Code(https://leetcode.com/submissions/detail/261926210/)
    时间复杂度：O(m * n)
    空间复杂度：O(1)
