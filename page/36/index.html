【二叉树专项】101.Symmetric Tree

一、题意

题目链接: 101.Symmetric Tree

1.Description

【输入】
给定一棵二叉树。
【输出】
判断这棵二叉树是否是对称的。
这里的对称是指：找一条穿过root节点的垂直线。然后反转，如果左右两侧完全重合则是对称的。

2.Example

Input: [1,2,2,3,4,4,3]
Output:
true

3.Corner Case

  1. 树为空时，true。

二、题解

Solution-1：Recursive solution(DFS)

1.调用辅助函数isMirror来判断两棵树是不是互为镜像。
那么本问题就转化为：当给定的树非空时，如果其左子树和其右子树互为镜像。那么给定的这棵树就是对称了。

2.考虑两棵树：node1和node2。
若node1和node2都是空，则互为镜像成立。
若node1和node2一个空，一个非空，则互为镜像不成立。
当node1和node2都不空时，如果其value不相等，则互为镜像不成立。
当node1和node2都不空时，且其value相等的情况下：
若node1的左子树和node2的右子树互为镜像，而且node1的右子树和node2的左子树互为镜像，则node1和node2互为镜像成立。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/275856697/)

Solution-2：Iterative solution using Queue(BFS)

1.当给定的树非空时，将root->left放入队列p，将root->right放入队列q。

2.循环，当两个队列同时非空的时候，每次从两个队列中取两个元素nodeP和nodeQ，检查：为null的情况，value的情况。
若同空，则continue
若一空一非空，直接return false
若均非空，但value不同，直接return false
若均非空，value相同，则：
对于队列p，先入nodeP的左儿子，然后右儿子。
对于队列q，陷入nodeQ的右儿子，然后左儿子。

以这样的顺序加入的话，我们可以使得每次取出的两个节点都是对称位置上的两个节点。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/288132559/)
