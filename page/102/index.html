48.Rotate Image

Posted on 2019-12-11

一、题意

题目链接: 48.Rotate Image

1.Description

给定一个n阶矩阵。
将这个矩阵顺时针旋转90°。
不得申请额外空间。即：in-place就地操作。

2.Example

Given input matrix =
[
[1,2,3],
[4,5,6],
[7,8,9]
],
rotate the input matrix in-place such that it becomes:
[
[7,4,1],
[8,5,2],
[9,6,3]
]

3.Corner Case

n: 当n为0的时候，直接返回false；

二、题解

Solution-1：Layer By Layer，One By One

1.n阶矩阵，需要考虑n / 2层，即层数：[0, n / 2 - 1]
2.对于每一层，j：[i, n - 2 - i]

时间复杂度：O(n ^ 2)
空间复杂度：O(1)

5.Code(https://leetcode.com/submissions/detail/285218920/)

Solution-2：

1.先将二维矩阵reverse，即保持其每一行中元素的顺序。
将这n行逆序。
[
[1,2,3],
[4,5,6],
[7,8,9]
]
reverse后：
[
[7,8,9],
[4,5,6],
[1,2,3]
]
2.此时可以发现，从[0, 0]到[n - 1, n - 1]的对角线上所有元素已经排列好了。
3.接下来，将对角线左右两侧的元素互换。
[
[7,8,9],
[4,5,6],
[1,2,3]
]
互换后：
[
[7,4,1],
[8,5,2],
[9,6,3]
]
4.此时已经完全排列好了，return即可。

时间复杂度：O(n ^ 2)
空间复杂度：O(1)
