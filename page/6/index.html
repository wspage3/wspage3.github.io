【二叉树专项】543.Diameter of Binary Tree

一、题意

题目链接: 543.Diameter of Binary Tree

1.Description

【输入】
给定一棵二叉树。
【输出】
求出这棵二叉树的直径。
二叉树的直径（Diameter of a Binary Tree）：树中任意两个节点构成的路径，最长的那条所包含的节点数目 - 1。
这条路径可以通过root，也可以不通过root。

2.Example

Input: [1,2,3,4,5,NULL,NULL]
Output: 3

3.Corner Case

  1. 当树为NULL的时候，直径为0
  2. 当树为单节点的时候，直径为1

二、题解

Solution-1：Calculate (Height + Length) using vector

1.对于一个节点node来说，以它为根的二叉树的直径，可能有以下两种情况：
a.直径经过node节点
b.直径不经过node节点
参考此图(https://www.cnblogs.com/miloyip/archive/2010/02/25/binary_tree_distance.html)

2.遍历每个节点，计算两个值：
a.以它为根的树的最大深度height
b.以它为根的树的直径length

3.由于这个节点的计算需要首先计算其左右子树，所以我们以dfs(Postorder)的方式遍历。
则题目的答案就是root节点对应的length。

4.边界值：
高度：这个节点到最底层的叶子节点的路径上，有几条边。
若一个节点为NULL，则其高度记为-1（方便公式的统一性），其直径记为0。
若一个节点为单节点，则其高度记为0，其直径记为0。

5.当其左、右子树的高度和直径都计算出来了。记为leftHeight, leftLength, rightHeight, rightLength。
则对于cur节点：
height = max(leftHeight, rightHeight) + 1;
length = max(leftHeight + rightHeight + 2, max(leftLength, rightLength));

时间复杂度：O(n)
空间复杂度：O(n)

6.Code(https://leetcode.com/submissions/detail/282673369/)

Solution-2：Calculate Maximun Length using member variable

0.遍历方向依旧是DFS。

1.maxDiameter以类成员的形式，不断进行更新

2.height以函数返回值的形式，不断进行更新

时间复杂度：O(n)
空间复杂度：O(n)
