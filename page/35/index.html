【DP专项】322.Coin Change

发表于 2020-01-03

一、题意

题目链接: 322.Coin Change

1.Description

【输入】
给定一个长度为n的整数数组。代表着你拥有这些面值的硬币。
给定一个正整数target。
【输出】
求出你最少需要多少个硬币，使得它们的钱的sum等于target。
如果拼不出来，则返回-1。

2.Example

Input: coins = [1, 2, 5], amount = 11
Output: 3
Explanation: 11 = 5 + 5 + 1

3.Corner Case

  1. 每种硬币可以使用无限次。
  2. 每种硬币的面值都大于0。那么当target小于0时，ans = -1。

二、题解

Solution-1：Dynamic Programming(Iterative)

  1. 思考：
    Q: What is important so far???
    A: The sum、minimum coins

  2. 状态定义：dp[i]：sum为i，所需要的硬币的最小个数。本题答案即dp[target]。

  3. 转移方程：dp[i] = min(dp[i], 1 + dp[i - x])。其中(i - x)大于等于0；x是nums中任一元素。

  4. 边界条件：dp[0] = 0；在计算dp[i]前，将其初始化为INT_MAX；

  5. 求解方向：
    【Backward DP】
    i从1开始，到target结束。通过dp[i - x]来计算求得dp[i]。

【Forward DP】
i从0开始，到target - 1结束。使用dp[i]来更新dp[i + x]。

  1. 空间优化：不能降维。

  2. Code(https://leetcode.com/submissions/detail/290651014/)
    时间复杂度：O(n * target)
    空间复杂度：O(target)

  3. Code(https://leetcode.com/submissions/detail/290653511/)
    时间复杂度：O(n * target)
    空间复杂度：O(target)

【DP专项】213.House Robber II

发表于 2020-01-02

一、题意

题目链接: 213.House Robber II

1.Description

【输入】
给定一个长度为n的整数数组。数组元素都是非负的。代表着n户人家的积蓄。
【输出】
计算出你能抢劫到最多多少钱。
不允许抢劫两个相邻的人家，会导致自动报警。
同时，这些人家绕成了一个圈。这意味着：第一户人家和最后一户人家是相邻的。

2.Example

Input: [2,3,2]
Output: 3
Explanation: You cannot rob house 1 (money = 2) and then rob house 3 (money = 2),
because they are adjacent houses.

3.Corner Case

  1. n为0时，答案为0。
  2. n为1时，答案为nums[0]。
  3. n为2时，答案为max(nums[0], nums[1])。

二、题解

Solution-1：Dynamic Programming

  1. 状态定义：dp[i]：抢劫完前i户([0, i])人家，最多能获得的钱。

  2. 转移方程：dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])

  3. 边界条件：dp[0] = nums[0], dp[1] = max(nums[0], nums[1])

  4. 求解顺序：
    i从0开始，到n - 2结束，结果记为ans1。
    i从1开始，到n - 1结束，结果记为ans2。
    答案即：max(ans1, ans2)。

  5. Code(https://leetcode.com/submissions/detail/272474951/)
    时间复杂度：O(n)
    空间复杂度：O(1)

不允许抢劫两个相邻的人家，会导致自动报警。

2.Example

Input: [2,7,9,3,1]
Output: 12
Explanation: Rob house 1 (money = 2), rob house 3 (money = 9) and rob house 5 (money = 1).Total amount you can rob = 2 + 9 + 1 = 12.

3.Corner Case

  1. n为0时，答案为0。
  2. n为1时，答案为nums[0]。
  3. n为2时，答案为max(nums[0], nums[1])。

二、题解

Solution-1：Dynamic Programming

  1. 状态定义：dp[i]：抢劫完前i户([0, i])人家，最多能获得的钱。

  2. 转移方程：dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])

  3. 边界条件：dp[0] = nums[0], dp[1] = max(nums[0], nums[1])

  4. 求解顺序：i从2开始，到n - 1结束。

  5. Code(https://leetcode.com/submissions/detail/272443961/)
    时间复杂度：O(n)
    空间复杂度：O(1)
