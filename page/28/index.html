【链表专项】024.Swap Nodes in Pairs

一、题意

题目链接: 024.Swap Nodes in Pairs

1.Description

【输入】
给定一个单链表。
【输出】
要求交换两两结点的顺序。如果仅有单节点，则无需处理。

2.Example

Given 1->2->3->4, you should return the list as 2->1->4->3.

3.Corner Case

  1. 链表为NULL时，无需处理。
  2. 链表为单节点时，无需处理。

二、题解

Solution-1：Iterative Solution

1.回忆【206.Reverse Linked List】的Solution-3，对于单链表[1->2->3->NULL]：
通过一次循环，将其转变为[2->1->3->NULL]。

2.用相似的思路进行处理：
使用dummy连接在head的前面。
pre指针初始化指向dummy。含义为：待swap的两个结点的前驱结点。

3.当pre以及pre->next->next均不为NULL的情况下，进行两两一对处理：
cur为两两一对的第一个结点。
next为两两一对的第二个结点。
swap后，pre向后移动两步。
最终返回dummy.next即可。

时间复杂度：O(n)
空间复杂度：O(1)

Solution-2：Recursive Solution

1.使用递归来解决：
边界条件：当子链表为单节点或者NULL时，不处理。

2.对于当前链表，头为head，过程如下：
首先求得子链表返回的结果：swapPairs(head->next->next); 将其接在head的后面。
再将head接在temp（原head的后继）的后面。

时间复杂度：O(n)
空间复杂度：O(n)
