【二叉树专项】226.Invert Binary Tree

Posted on 2019-12-24

一、题意

题目链接: 226.Invert Binary Tree

1.Description

【输入】
给定一棵二叉树。
【输出】
将其翻转。
也就是求出与它互为镜像的二叉树。

2.Example

Input: [4, 2, 7, 1, 3, 6, 9]
Output: [4, 7, 2, 9, 6, 3, 1]

3.Corner Case

  1. 树为NULL时，返回NULL。
  2. 树仅包含一个node时，返回这个节点。

二、题解

Solution-1：Recursive solution(DFS)

1.遍历顺序为dfs的顺序

2.对于任一节点来说：
通知两儿子（递归调用，使得两儿子分别进行invert操作）
自己计算（将自己的左儿子更新为：右儿子invert后返回的值；将右儿子更新为：左儿子invert后返回的值）
上报给父亲（返回自己）

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/275865590/)

Solution-2：Iterative solution using Queue(BFS)

1.遍历顺序为bfs的顺序

2.每次遍历到一个节点node时，将其左右儿子交换。然后再将其放入队列。
这样的话，每个节点的左右儿子都被交换过。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/275868273/)
