【DP专项】062.Unique Paths

一、题意

题目链接: 062.Unique Paths

1.Description

【输入】
给定两个整数m、n。代表着一个m * n的矩阵。
【输出】
求出，从left-top到right-bottom，一共有多少种不同的路径。
在任意位置上，只能往下或者往右走一步。

2.Example

Input: m = 3, n = 2（3列2行）
Output: 3
Explanation:
From the top-left corner, there are a total of 3 ways to reach the bottom-right corner:

  1. Right -> Right -> Down
  2. Right -> Down -> Right
  3. Down -> Right -> Right

3.Corner Case

  1. m或者n为0时，答案为0。
  2. m或者n为1时，答案为1。

二、题解

Solution-1：Dynamic Programming

  1. 状态定义：dp[i][j]：到达位置(i,j)上，有多少种不同的方法。

  2. 转移方程：dp[i][j] = dp[i - 1][j] + dp[i][j - 1]

  3. 边界条件：当i或者j为0时，dp[i][j] = 1

  4. 求解顺序：从(0,0)开始计算，到(m - 1, n - 1)停止。

  5. 优化：二维数组降为一维数组。

6.Code(https://leetcode.com/submissions/detail/261628264/)
时间复杂度：O(m * n)
空间复杂度：O(m * n)
