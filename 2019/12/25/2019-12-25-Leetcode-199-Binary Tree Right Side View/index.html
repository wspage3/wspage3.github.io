【二叉树专项】199.Binary Tree Right Side View

一、题意

题目链接: 199.Binary Tree Right Side View

1.Description

【输入】
给定一棵二叉树，想象你站在它的右侧。
【输出】
输出你能看到的数字，从树的top到bottom，结果存在一维数组中。

2.Example

Input: [1,2,3,null,5,null,4]
Output: [1, 3, 4]

3.Corner Case

  1. 树为NULL时，返回空数组。

二、题解

分析

本题实质就是求：树的每一层中，最右边的那个元素。如果树有k层，则答案中就包含k个数据。

Solution-1：Recursive Solution(DFS)

1.回想【102.Binary Tree Level Order Traversal】，我们同样也是采用recursive的dfs来遍历树：
如果是第一次进入该level，则新建一个sub数组存储在该level上的节点。遍历顺序：根 -> 左儿子 -> 右儿子

2.在本题中，采用根 -> 右儿子 -> 左儿子的遍历顺序。使得：第一次进入某level的就是最右边的那个节点。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276672883/)

Solution-2：Iterative Solution using Queue

1.回想【102.Binary Tree Level Order Traversal】的迭代版本，我们使用了队列：
每一轮，放入同一层中的所有节点，将其统一加入一个sub一维数组中，再将这个一维数组加入到二维答案中。

2.本题中，我们同样采用这样的遍历方法，当每一轮中访问同一层的所有节点时，我们将最后的那个节点放入ans中即可。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276668971/)
