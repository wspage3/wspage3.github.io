【BST专项】098.Validate Binary Search Tree

发表于 2019-12-25

一、题意

题目链接: 098.Validate Binary Search Tree

1.Description

【输入】
给定一棵树。
【输出】
判断其是否为一棵BST(Binary Search Tree)(Binary Sort Tree)
【背景】
BST（二叉搜索树）（二叉排序树）
定义：
a.若左子树非空，左子树中所有节点的值都比根节点的值小。
b.若右子树非空，右子树中所有节点的值都比根节点的值大。
c.左子树、右子树都是BST。
特点：
a.树中任一节点的值是唯一的。
b.中序遍历序列是一个严格升序序列。

2.Example

Input: [5,1,4,null,null,3,6]
Output: false
Explanation: 节点4在根节点5的右子树，矛盾。

3.Corner Case

  1. 树为NULL时，true。
  2. 树为单节点时，true。

二、题解

Solution-1：Recursive Solution(Preorder)

1.遍历顺序为preorder。

2.从root开始，依次dfs遍历到每一个节点
对于当前节点node来说：
我的父亲调用我的时候，把他期望我所在的取值范围告诉我：(left, right)
首先：我判断自己是否在这个范围内，若不在，说明我的val不满足BST的性质
其次：我需要判断左子树：我期望其在以下取值范围内：(left, node->val)
右子树：我期望其在以下取值范围内：(node->val, right)
如果以上都满足，则开开心心的返回true。有哪一环不满足，都返回false。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276727177/)

Solution-2：Iterative Solution(Inorder)

1.遍历顺序为inorder。

2.每次比较当前值和前一个值的大小。只有当前值严格大于前一个值，才符合BST的性质。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276738088/)
