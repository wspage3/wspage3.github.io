【二叉树专项】110.Balanced Binary Tree

一、题意

题目链接: 110.Balanced Binary Tree

1.Description

【输入】
给定一棵二叉树。
【输出】
判断其是否是一棵“高度平衡的二叉树”
“高度平衡的二叉树”：树中所有的节点，其左子树和右子树的高度差不大于1。

2.Example

Input: [3,9,20,null,null,15,7]
Output: true

Input: [1,2,2,3,3,null,null,4,4]
Output: false

3.Corner Case

  1. 树为NULL时，return true
  2. 树为单节点时，return true

二、题解

Solution-1：Top-to-Bottom(Dual Recursive)

1.自顶向下，分别判断每个节点的左右子树的高度，以及该左右子树是否为balanced tree。

2.缺点：双递归的存在，使得时间效率不高。计算高度存在大量重复计算。

时间复杂度：O(n)
空间复杂度：O(n)

2.Code(https://leetcode.com/submissions/detail/276397293/)

Solution-2：Bottom-to-Top(Single Recursive)

1.自底向上，采用dfs的遍历路径（Postorder）。
对于遍历到的每一个节点，需要做以下事情：
a.询问我的左儿子是否为balanced tree，若是的话让他返回其高度，不是的话返回-1
b.询问我的右儿子是否为balanced tree，若是的话让他返回其高度，不是的话返回-1
c.现在我知道自己的两个儿子的情况了，我可以判断出自己是否为balanced tree。我判断出来，并将其告诉我的父亲

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276435936/)
