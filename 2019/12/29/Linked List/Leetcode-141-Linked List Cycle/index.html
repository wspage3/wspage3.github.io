【链表专项】141.Linked List Cycle

一、题意

题目链接: 141.Linked List Cycle

1.Description

【输入】
给定一个单链表，头节点为head。
【输出】
判断上述链表是否有环，有返回true，无返回false。

2.Example

Input: head = [3,2,0,-4], pos = 1
Output: true
Explanation: There is a cycle in the linked list, where tail connects to the second node.

3.Corner Case

  1. 如果链表为NULL，则false。
  2. 如果链表为单节点，则false。
  3. 要求O(1)的space complexity。

二、题解

Solution-1：Fast and Slow Pointers

1.要想形成环，则链表中至少需要两个节点。

2.使用快慢指针法：两指针同时从head出发，slow每次走一步，fast每次走两步。
如果某个时间，它们相遇了，则说明链表中一定有环。

3.当fast以及fast->next都不为NULL的时候，fast才能向后移动到下一个位置。
否则会发生访问地址错误。

时间复杂度：O(n)
空间复杂度：O(1)
