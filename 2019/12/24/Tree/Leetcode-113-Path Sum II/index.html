【二叉树专项】113.Path Sum II

发表于 2019-12-24 更新于 2019-12-26

一、题意

题目链接: 113.Path Sum II

1.Description

【输入】
给定一棵二叉树。
给定一个整数target。
【输出】
判断是否有一条从root到leaf的路径，这条路径上所有节点的sum为target。
如果存在，则返回一个二维数组，代表有n条路径满足题意，每一条路径是一个一维数组。
如果不存在，返回空的二维数组。

2.Example

Input: 22
[5,4,8,11,null,13,4,7,2,null,null,5,1]
Output:
[
[5,4,11,2],
[5,8,4,5]
]

3.Corner Case

  1. 树为NULL时，返回空二维数组。
  2. 树仅包含一个node时，查看node->val 是否等于target。

二、题解

Solution-1：Recursive solution(DFS)(Backtracking)

1.遍历顺序为dfs的顺序

2.对于任一节点来说：
父亲调用自身的时候：
a.传递一个值target下来。
b.传递一个路径下来，存在一维数组sub中。

3.叶节点需要做：
首先，将自己放在路径中。
其次，判断自身（如果自身value和target相等，那么将这条路径作为一个合法的子答案加入res；如果不相等的话，不加入ans）
然后，删除自身。因为本节点已经被考虑完成了，也不可能有后继节点了。所以从sub中删除本节点。
最后，上报父亲。return即可。

4.非叶节点需要做：
首先，将自己放在路径中。
其次，判断自身（自身一定不满足，因为不是叶子）
然后，判断非空的儿子。将target - node->val传递下去。
接着，删除自身。因为本节点自身不满足，而其非空儿子都考虑完成了。所以从sub中删除本节点。
最后，上报父亲。return即可。

时间复杂度：O(n)
空间复杂度：O(n)
