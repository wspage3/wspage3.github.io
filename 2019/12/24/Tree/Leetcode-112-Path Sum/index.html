【二叉树专项】112.Path Sum

一、题意

题目链接: 112.Path Sum

1.Description

【输入】
给定一棵二叉树。
给定一个整数target。
【输出】
判断是否有一条从root到leaf的路径，这条路径上所有节点的sum为target。
存在返回true，不存在返回false。

2.Example

Input: 22
[5,4,8,11,null,13,4,7,2,null,null,null,1]
Output: true(5->4->11->2)

3.Corner Case

  1. 树为NULL时，返回false。
  2. 树仅包含一个node时，return node->val == target ? true : false。

二、题解

Solution-1：Recursive solution(DFS)

1.遍历顺序为dfs的顺序

2.对于任一节点来说：
父亲调用自身的时候，传递一个值target下来。
判断自身（如果自身是叶子节点的话，直接判断自身value是否和target相等）
判断两儿子（递归调用，将target - node->val传递给左右儿子）
上报给父亲（返回结果）

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276069188/)

Solution-2：Iterative solution using Stack(DFS-Preorder Traversal)

1.遍历顺序为dfs的顺序

2.开两个stack，一个用来遍历node，一个用来存相同位置上的node对应的target。
使用iterative版本preorder的顺序遍历每一个node。
每次从两个栈顶pop：考察一个node，考察一个value。
如果该node是leaf的话，对其进行检查：如果刚刚抛出的value为0，则说明存在一条满足题意的路径。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276075062/)
