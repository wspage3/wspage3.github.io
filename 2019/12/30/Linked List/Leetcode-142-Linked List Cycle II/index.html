【链表专项】142.Linked List Cycle II

发表于 2019-12-30

一、题意

题目链接: 142.Linked List Cycle II

1.Description

【输入】
给定一个单链表，头节点为head。
【输出】
判断上述链表是否有环，有返回环开始的结点，无返回NULL。

2.Example

Input: head = [3,2,0,-4], pos = 1
Output: tail connects to node index 1
Explanation: There is a cycle in the linked list, where tail connects to the second node.

3.Corner Case

  1. 如果链表为NULL，则NULL。
  2. 如果链表为单节点，则NULL。
  3. 要求O(1)的space complexity。

二、题解

Solution-1：Fast and Slow Pointers

3.假设存在环路，有两种可能：
a.链表的tail连接到链表的head，构成一个大环
b.链表的tail连接到链表中部的某个node，构成一个线段以及一个小环。
4.考虑有环路的情况，slow和fast初始点都为head，slow每次走一步，fast每次走两步。slow最终一定会和fast相遇。
假设head到cycleStart的距离为s1，cycleStart到meet点的距离为s2，meet点返回cycleStart的距离为s3。
slow走过的距离为s(slow)：s1 + s2
fast走过的距离为s(fast)：s1 + s2 + k * (s3 + s2)。k >= 1。
当k为1的时候，s1 == s3
当k为2的时候，s1 == s2 + 2s3 = s3 + (s2 + s3)
当k为3的时候，s1 == 2s2 + 3s3 = s3 + 2(s2 + s3)
从而根据这个特点找到cycleStart的位置。
