【二叉树专项】113.Path Sum II

一、题意

题目链接: 113.Path Sum II

1.Description

【输入】
给定一棵二叉树。
给定一个整数target。
【输出】
判断是否有一条从root到leaf的路径，这条路径上所有节点的sum为target。
如果存在，则返回一个二维数组，代表有n条路径满足题意，每一条路径是一个一维数组。
如果不存在，返回空的二维数组。

2.Example

Input: 22
[5,4,8,11,null,13,4,7,2,null,null,5,1]
Output:
[
[5,4,11,2],
[5,8,4,5]
]

3.Corner Case

  1. 树为NULL时，返回空二维数组。
  2. 树仅包含一个node时，查看node->val 是否等于target。

二、题解

Solution-1：Recursive solution(DFS)

1.遍历顺序为dfs的顺序

2.对于任一节点来说：
父亲调用自身的时候：
a.传递一个值target下来。
b.传递一个路径下来，存在一维数组sub中。

3.叶节点需要做：
首先，将自己放在路径中。
其次，判断自身（如果自身value和target相等，那么将这条路径作为一个合法的子答案加入res；如果不相等的话，不加入ans）
然后，删除自身。因为本节点已经被考虑完成了，也不可能有后继节点了。所以从sub中删除本节点。
最后，上报父亲。return即可。

4.非叶节点需要做：
首先，将自己放在路径中。
其次，判断自身（自身一定不满足，因为不是叶子）
然后，判断非空的儿子。将target - node->val传递下去。
接着，删除自身。因为本节点自身不满足，而其非空儿子都考虑完成了。所以从sub中删除本节点。
最后，上报父亲。return即可。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276102013/)

【二叉树专项】112.Path Sum

一、题意

题目链接: 112.Path Sum

1.Description

【输入】
给定一棵二叉树。
给定一个整数target。
【输出】
判断是否有一条从root到leaf的路径，这条路径上所有节点的sum为target。
存在返回true，不存在返回false。

2.Example

Input: 22
[5,4,8,11,null,13,4,7,2,null,null,null,1]
Output: true(5->4->11->2)

3.Corner Case

  1. 树为NULL时，返回false。
  2. 树仅包含一个node时，return node->val == target ? true : false。

二、题解

Solution-1：Recursive solution(DFS)

1.遍历顺序为dfs的顺序

2.对于任一节点来说：
父亲调用自身的时候，传递一个值target下来。
判断自身（如果自身是叶子节点的话，直接判断自身value是否和target相等）
判断两儿子（递归调用，将target - node->val传递给左右儿子）
上报给父亲（返回结果）

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276069188/)

Solution-2：Iterative solution using Stack(DFS-Preorder Traversal)

1.遍历顺序为dfs的顺序

2.开两个stack，一个用来遍历node，一个用来存相同位置上的node对应的target。
使用iterative版本preorder的顺序遍历每一个node。
每次从两个栈顶pop：考察一个node，考察一个value。
如果该node是leaf的话，对其进行检查：如果刚刚抛出的value为0，则说明存在一条满足题意的路径。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276075062/)

【二叉树专项】226.Invert Binary Tree

一、题意

题目链接: 226.Invert Binary Tree

1.Description

【输入】
给定一棵二叉树。
【输出】
将其翻转。
也就是求出与它互为镜像的二叉树。

2.Example

Input: [4, 2, 7, 1, 3, 6, 9]
Output: [4, 7, 2, 9, 6, 3, 1]

3.Corner Case

  1. 树为NULL时，返回NULL。
  2. 树仅包含一个node时，返回这个节点。

二、题解

Solution-1：Recursive solution(DFS)

1.遍历顺序为dfs的顺序

2.对于任一节点来说：
通知两儿子（递归调用，使得两儿子分别进行invert操作）
自己计算（将自己的左儿子更新为：右儿子invert后返回的值；将右儿子更新为：左儿子invert后返回的值）
上报给父亲（返回自己）

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/275865590/)

Solution-2：Iterative solution using Queue(BFS)

1.遍历顺序为bfs的顺序

2.每次遍历到一个节点node时，将其左右儿子交换。然后再将其放入队列。
这样的话，每个节点的左右儿子都被交换过。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/275868273/)

【二叉树专项】111.Minimum Depth of Binary Tree

一、题意

题目链接: 111.Minimum Depth of Binary Tree

1.Description

【输入】
给定一棵二叉树。
【输出】
找出其最小的深度。
最小深度也就是求：从root到所有的leaf中，路径上最少包含几个节点。

2.Example

Input: [3,9,20,null,null,15,7]
Output: 2

3.Corner Case

  1. 树为空时，0。
  2. 树仅包含一个node时，1。

二、题解

Solution-1：Recursive solution(DFS)

1.遍历顺序为dfs的顺序

2.对于任一节点来说：
询问两儿子（递归调用）
计算出来（计算两个答案的最小值，再加一）
上报给父亲（返回答案）

3.本题目中和[Leetcode-104]不同的地方在于：
若一个节点p非空，且它两个儿子一个空（子答案0），一个非空时（子答案x）：
对于节点p来说，不能用min(0, x) + 1来计算。而应该是：x + 1

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276161504/)

Solution-2：Iterative solution using Queue(BFS)

1.遍历顺序为bfs的顺序

2.如果遍历到某一层，发现此时某个节点为叶节点，则直接返回当前的层数cnt。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/288166913/)

【二叉树专项】101.Symmetric Tree

一、题意

题目链接: 101.Symmetric Tree

1.Description

【输入】
给定一棵二叉树。
【输出】
判断这棵二叉树是否是对称的。
这里的对称是指：找一条穿过root节点的垂直线。然后反转，如果左右两侧完全重合则是对称的。

2.Example

Input: [1,2,2,3,4,4,3]
Output:
true

3.Corner Case

  1. 树为空时，true。

二、题解

Solution-1：Recursive solution(DFS)

1.调用辅助函数isMirror来判断两棵树是不是互为镜像。
那么本问题就转化为：当给定的树非空时，如果其左子树和其右子树互为镜像。那么给定的这棵树就是对称了。

2.考虑两棵树：node1和node2。
若node1和node2都是空，则互为镜像成立。
若node1和node2一个空，一个非空，则互为镜像不成立。
当node1和node2都不空时，如果其value不相等，则互为镜像不成立。
当node1和node2都不空时，且其value相等的情况下：
若node1的左子树和node2的右子树互为镜像，而且node1的右子树和node2的左子树互为镜像，则node1和node2互为镜像成立。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/275856697/)

Solution-2：Iterative solution using Queue(BFS)

1.当给定的树非空时，将root->left放入队列p，将root->right放入队列q。

2.循环，当两个队列同时非空的时候，每次从两个队列中取两个元素nodeP和nodeQ，检查：为null的情况，value的情况。
若同空，则continue
若一空一非空，直接return false
若均非空，但value不同，直接return false
若均非空，value相同，则：
对于队列p，先入nodeP的左儿子，然后右儿子。
对于队列q，陷入nodeQ的右儿子，然后左儿子。

以这样的顺序加入的话，我们可以使得每次取出的两个节点都是对称位置上的两个节点。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/288132559/)

【二叉树专项】100.Same Tree

一、题意

题目链接: 100.Same Tree

1.Description

【输入】
给定两棵二叉树。
【输出】
判断这两棵二叉树是否是同一个。

2.Example

Input: [1,2], [1,null,2]
Output:
false

3.Corner Case

  1. 两树都为空时，true。
  2. 两树其中一个为空时，false。
  3. 然后考虑两树都不为空的场景。

二、题解

Solution-1：Recursive solution(DFS)

1.当两树p, q都不为空的情况下：
首先，判断节点q和节点q的value是否相等，不相等则return false。
其次，递归判断p和q的左子树是否same，以及p和q的右子树是否same。

时间复杂度：O(n)
空间复杂度：O(n) //最坏n（完全不平衡二叉树），最好log n（完全平衡二叉树）

本质顺序：先序遍历的recursive版本。

2.同样的,也可以使用iterative + stack的版本来对两棵树同时进行前序/中序/后序遍历.
遍历的过程中进行比较.

3.Code(https://leetcode.com/submissions/detail/275771314/)

Solution-2：Iterative solution using Queue(BFS)

1.按照bfs的顺序,将p和q所有的节点入队列.
每次取出一个元素进行比较.
如果比较通过,则将它们的左右儿子同时放入各自的队列中.

2.若两个树same的话,中途不会return false.且最终两个队列里都为空.

本质:层次遍历的iterative版本

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/275800085/)

【二叉树专项】103.Binary Tree Zigzag Level Order Traversal

一、题意

题目链接: 103.Binary Tree Zigzag Level Order Traversal

1.Description

【输入】
给定一棵二叉树。
【输出】
求出这棵二叉树的Zigzag的level序遍历结果，并以二维数组的形式返回。
垂直顺序：从根往叶。
水平顺序：先从左往右，后从右往左。交替进行。

2.Example

Input: [3,9,20,null,null,15,7]
Output:
[
[3],
[20,9],
[15,7]
]

3.Corner Case

  1. 树为空时，返回空数组。

二、题解

Solution-1：Recursive solution + Reverse

1.和上题相比，垂直顺序不变。
水平顺序变成：依次从左往右，从右往左。
将遍历出来的ans[i]进行Reverse。i = 1，3，5…

时间复杂度：O(n)
空间复杂度：O(n)

2.Code(https://leetcode.com/submissions/detail/276450407/)

Solution-2：Iterative solution using Queue + Reverse

1.当计算出res[i]，准备将其放入ans的时候，检查：若i是奇数，则reverse。
2.当一轮中往res[i]中添加同一层的元素时，检查：若i时奇数，则sub[n - 1 - i] = node->val。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276462285/)

Solution-3：Iterative solution using Deque

1./* zig（正常）：push_back, front, pop_front, 先left后right /
/ zag（反序）：push_front, back, pop_back, 先right后left */

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/276467788/)

【二叉树专项】102.Binary Tree Level Order Traversal

一、题意

题目链接: 102.Binary Tree Level Order Traversal

1.Description

【输入】
给定一棵二叉树。
【输出】
求出这棵二叉树的level序遍历结果，并以二维数组的形式返回。
从根往叶，从左往右。

2.Example

Input: [3,9,20,null,null,15,7]
Output:
[
[3],
[9,20],
[15,7]
]

3.Corner Case

  1. 树为空时，返回空数组。

二、题解

Solution-1：Recursive solution

0.前序、中序、后序：都属于dfs。当实现迭代版本时，我们使用的都是stack。
1.层次遍历：依旧用dfs实现。

时间复杂度：O(n)
空间复杂度：O(n)

2.Code(https://leetcode.com/submissions/detail/275764307/)

Solution-2：Iterative solution using Queue

0.前序、中序、后序：都属于dfs。当实现迭代版本时，我们使用的都是Stack模拟。
1.层次遍历：实现迭代版本时，使用bfs，用Queue模拟。
2.每一轮，从队列中依次拿出n个元素，这些元素一定是同一层的。将这个结果放入ans中。
拿的时候，同时考虑其左右儿子，如果有的话，将其加入队列中。
3.直到队列为空的时候，所有的元素都被考虑过了。

时间复杂度：O(n)
空间复杂度：O(n)

3.Code(https://leetcode.com/submissions/detail/275767633/)
